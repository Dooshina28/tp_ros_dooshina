3. rosrun tp_ros_dooshina sub.py
Rien ne se passe , rien n'est afficher
ouvrir un autre terminal
4. rosrun tp_ros_dooshina pub.py
Les commandes de pub.py sont afficher sur le terminal ou sub.py a ete lancer
